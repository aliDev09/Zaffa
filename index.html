Tab 1

HTML

  • HTML — это стандартизированный язык разметки для отображения документов в браузере. Мы изучим его, чтобы уметь размещать на веб-странице текст, изображения, таблицы и видео.

CSS

  • CSS — это язык стилей, который позволяет задавать внешний вид и оформление элементов веб-страницы. Мы научимся стилизовать страницы, делая их красивыми и удобными для пользователя.

Figma

  • Figma — это онлайн-инструмент для создания интерфейсов и прототипов. Мы будем использовать его для разработки макетов веб-страниц и приложений

Tab 2

Scss

  • SCSS — это расширение языка CSS, которое добавляет мощные возможности, такие как переменные, вложенные правила и миксины. Мы изучим, как с помощью SCSS писать более организованный и переиспользуемый CSS-код, что ускорит и упростит процесс создания стилей для веб-страниц.

Bootstrap

  • Bootstrap — это популярный CSS-фреймворк, который позволяет быстро создавать адаптивные веб-страницы. Мы изучим, как использовать готовые компоненты для ускорения разработки.

GIT

  • Git — это система контроля версий, которая помогает отслеживать изменения в коде. Мы научимся работать с ветвями, коммитами и слиянием изменений.

GIT HUB

  • GitHub — это платформа для хранения и совместной работы над проектами с использованием Git. Мы изучим, как размещать и управлять проектами на GitHub.

Tab 3

JAVASCRIPT

  • JavaScript — это язык программирования, который позволяет создавать интерактивные элементы на веб-страницах. Мы изучим основные концепции, такие как переменные, функции и события, чтобы добавить динамику на сайты.

Jquery

  • jQuery — это библиотека JavaScript, которая упрощает написание кода и улучшает кроссбраузерную совместимость. Мы изучим, как с помощью jQuery выбирать элементы, добавлять к ним события и создавать динамические страницы.

Jquery data table

  • jQuery DataTables — это мощный плагин для упрощенного управления HTML-таблицами. Мы изучим, как с помощью этого плагина добавлять к таблицам функции поиска, сортировки и постраничной навигации.

Gsap

  • GSAP — это мощная библиотека для создания плавных и сложных анимаций на веб-страницах. Мы научимся анимировать элементы, работать с временными шкалами и добавлять интерактивные эффекты.

Chart js

  • Chart.js — это простая в использовании библиотека для создания интерактивных графиков и диаграмм. Мы изучим, как с помощью Chart.js визуализировать данные и настраивать различные типы графиков, такие как линейные, круговые и столбчатые диаграммы.

Codepen ui

  • CodePen — это популярная платформа для фронтенд-разработчиков, где можно быстро создавать, тестировать и делиться своими проектами. Мы разберем интерфейс CodePen и научимся эффективно использовать его для демонстрации и разработки веб-решений.
